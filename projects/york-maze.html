ⓘ OVERVIEW

I developed a workflow and Unreal Engine for Fortnite (UEFN) project for James Afshar Dodson at Pop Up VR. The project is a promotional virtual version of the York Maze that can be played on Fortnite Creative by anyone.

⌨ DEVELOPMENT

The main challenge of this project was developing an easy, repeatable workflow for annually updating the virtual maze to match its real-world counterpart. I outlined the best methods I found for converting York Maze's initial 2D line art image into a 3D corn maze using Photoshop and Blender. This works by using Blender's Displace modifier to convert the image to a mesh, then using the mesh as a volume for Unreal Engine's foliage tool to populate cornstalks.
