⌨ DEVELOPMENT

I developed this website using a mix of Bootstrap and vanilla CSS. I created a "libraries" system that creates collections of project cards by criteria like their genre and tags. Each project's basic info is stored in JSON and used to create project cards and sections of each project's page.
