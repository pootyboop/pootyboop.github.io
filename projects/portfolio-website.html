ⓘ OVERVIEW

elliotgmann.com - the website you're on now! I upload my projects here including games, music, jobs, and other fun stuff.

⌨ DEVELOPMENT

I developed this website using a mix of Bootstrap and vanilla CSS. I created a "libraries" system that creates collections of project cards by criteria like their genre and tags. Each project's basic info is stored in JSON and used to create project cards and sections of each project's page.

I used a high contrast color scheme, alt tags, and other accessibility features to make the website easy to use. The website is fully responsive and can be used on mobile with no issues (except for playing web games, which are desktop exclusive). I added features like a Similar Projects library at the bottom of each project, Featured and Latest project libraries on the home page, and a searchbar to keep users engaged and help them find the content they want.

The searchbar creates a custom library by filtering projects by their skills, platforms, and other tags. For example, you can search for "Games", "Unreal Engine", "Sandboard", or any fragment of one of those terms (e.g. "sandbo") to find what you're looking for. You can also click any skill shown in each project's "Development" section to search for other projects where I used that same skill.
