RESOLUTION
This project may be marked at a resolution of 1280x800 or 1920x1080 with no change to the user experience.

PLUGINS
This project uses the P5JS plugin. This plugin is included in the directory and does not need to be installed.
!!! For P5JS to work, you will not be able to directly run the HTML file in Google Chrome. !!!
Instead, open the folder in Visual Studio Code and preview index.html with the Live Preview plugin.
This will work with an external browser (recommended) or the embedded preview.
